# UPSRaspberry

###Description
This is project of UPS for Raspberry Pi 2 based on TP4056 charging module and ATtiny13.
Idea from __hardlock.org.ua__ [1S li-ion indicator](http://hardlock.org.ua/viewtopic.php?f=9&t=442).
